Authentication

GET /v1/auth/signUp

Endpint que cadastra o usuário. O usuário não deve ter email repedito e a senha deve conter 8 caracteres com um especial uma letra maúscula com minúscula e um número.

Requisição


{
  "email": "jhon@gmail.com",
  "password": "#Minhasenha123"
}

POST /v1/auth/signIn

Se autenticar na plataforma.

A o usuário deve já estar cadastrado, e a senha deve deve conter 8 caracteres com um especial uma letra maúscula com minúscula e um número.

            {
